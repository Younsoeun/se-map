# 기능 요청 체크리스트

이 대화에서 요청한 기능들을 기록해두는 곳입니다. 구현이 끝나면 체크하고, 검증까지 마치면 함께 표시하세요.

## 원본 스펙 (`유럽 여행 지도 프롬프트.txt`)

- [x] 세계지도 → 국가 상세(포르투갈) 2단계 화면, 실제 Natural Earth 지리 좌표 기반 SVG
- [x] 포르투갈: 12개 도시, 28개 명소 (역사/소개/방문팁 3개 필드, 5개 카테고리)
- [x] 국가 상세 화면: admin-1 행정구역 경계 표시, 도시 점 마커 클릭 시 스무스 스크롤
- [x] 명소 이름 검색창 + 카테고리 필터 pill
- [x] 라이트/다크 테마 (`prefers-color-scheme` + `data-theme` 오버라이드)

## 이번 대화에서 추가 요청한 기능

- [x] **방문 여부 체크박스**: 각 플레이스(명소)별로 방문했는지 표시할 수 있는 체크박스 — `index.html`의 각 명소 카드
- [x] **방문 소감 기록**: 각 플레이스별로 짧은 방문 소감/메모를 남길 수 있는 텍스트 입력 — 체크박스 아래 textarea
- [x] **지도 색칠**: 방문한 곳은 지도 위 마커 색이 다르게 표시됨 (미방문/일부방문/전체방문 3단계) — `js/map-app.js`의 `styleCityMarker`
- [x] **국가 상세 통계에 방문 현황 추가**: "방문 X/28 (Y%)" 형태로 기존 통계 줄에 추가 — `#country-stats`
- [x] **문화 페이지 신설** (`culture.html`): 도시 단위가 아니라 **국가 단위**로 관리
  - [x] 국가별 특색 있는 문화·특징 정리 (한눈에 보기 / 인사·언어 예절 / 음식 문화 / 사회적 매너·터부 / 공휴일·축제 / 여행자를 위한 팁)
  - [x] 포르투갈부터 작성, 이후 국가 추가하기 쉬운 구조로 설계 — `js/culture-data.js`의 `countries` 배열에 추가만 하면 됨
  - [x] 지도 페이지와 동일한 디자인 시스템(그라디언트 블롭/pill/폰트) 적용
- [x] **디자인 참고**: mycalmsite.com 스타일 반영 → 상세 내용은 [design-references.md](design-references.md) 참고

## 저장 방식 관련 결정 사항

- [x] 방문 여부/소감은 `localStorage`에 저장 (백엔드 없음, 기기·브라우저별로 별도 유지) — `js/visited.js`
- [x] **내보내기(Export) 기능**: 방문 기록을 JSON 파일로 다운로드 — 상단 내비게이션 "내보내기" 버튼
- [x] **가져오기(Import) 기능**: 내보낸 JSON 파일을 업로드해 다른 기기/브라우저에서 복원 — 상단 내비게이션 "가져오기" 버튼

## 범위 관련 결정 사항

- [x] 1차 구현 범위는 포르투갈만 (원본 스펙의 "포르투갈 100% 완성 후 확장" 방침 유지)
- [x] 문화 페이지도 동일하게 포르투갈부터 시작, 다른 국가는 이후 같은 템플릿으로 추가
