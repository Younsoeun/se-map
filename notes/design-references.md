# 디자인 참고 자료 기록

이 파일은 제작 과정에서 참고한 사이트들과, 그중 마음에 든 요소·이유를 기록해두는 곳입니다.
나중에 실제 결과물에 잘 반영됐는지 확인하는 용도로 사용하세요.

## 버튼 형태 (첨부 이미지 IMG_6963 — 'Air Max 90 / + Add' 카드)

**마음에 든 요소** (사용자 직접 언급): 요소 테두리의 굴곡(완전 pill이 아닌 부드러운 라운드 사각형), 텍스트와 테두리 사이의 넉넉한 여백.

**반영 결정 (2026-07-11)**: 버튼·태그·내비게이션을 완전 pill(`border-radius: 999px`)에서 **부드러운 라운드 사각형**으로 전환. 적용값:
- `.pill`: `border-radius: 13px`, `padding: 9px 18px` (여백 넉넉히)
- `.nav-links`: 컨테이너 `15px`, 링크 `10px` / `padding: 9px 18px`
- `.route-badge`(확정·선택 태그, 이미지의 'Running' 태그처럼): `border-radius: 8px`
- 카드(`.panel`, 명소·문화·루트 카드)는 기존 16~20px 유지 — 이미지의 바깥 카드 굴곡과 동일 계열
- 원형 아이콘 버튼(테마 토글)은 그대로 원형 유지

## mycalmsite.com

**링크**: https://mycalmsite.com/

**마음에 든 요소** (사용자가 직접 언급): 배경 색의 그라디언트, 요소의 색감 및 형태

실제 사이트를 라이트/다크 모드로 열어 스크린샷과 computed style을 확인한 결과:

- **배경**: 완전 단색이 아니라, 큰 원형 `radial-gradient(circle, color 0%, transparent 72%)` 블롭 3~4개를 절대 위치로 겹쳐서 은은하게 번지는 느낌을 만듦. 라이트 모드는 연보라/연핑크/하늘색/민트 계열, 다크 모드는 같은 블롭이 짙은 배경 위에서 보라/자홍빛 광원처럼 보임.
- **버튼/탭 형태**: 완전 pill 형태 (`border-radius: 50px`) — 시간 선택 탭, "Begin" 버튼 등 전부.
- **그림자/글로우**: 주요 CTA 버튼에만 아주 은은한 glow (`0 0 32px rgba(255,255,255,0.08)` 수준, 하드한 네온 아님).
- **폰트**: Poppins 계열 (지오메트릭 산세리프).
- **분위기**: 차분하고 여백이 넉넉함, 장식 요소(중앙의 유리질 구체)는 이번 프로젝트 성격과 맞지 않아 채택하지 않음.

**최종 반영 결정**: 기존 프롬프트(`유럽 여행 지도 프롬프트.txt`)의 웜그레이/네이비/테라코타 팔레트는 그대로 유지하고, mycalmsite에서는 ① radial-gradient 블롭 배경 기법, ② 완전 pill 버튼/태그 형태, ③ 주요 CTA에 한정된 아주 은은한 glow — 이 세 가지 기법만 가져와 기존 팔레트 색상으로 다시 그림. mycalmsite의 라벤더/핑크 색상 자체는 채택하지 않음 (기존 프롬프트가 "무거운 그림자·글로우 pill 버튼"을 명시적으로 피하라고 했기 때문에, 형태만 차용하고 무게감은 절제).

## 기존 프롬프트에 이미 명시된 참고 사이트 3곳

`유럽 여행 지도 프롬프트.txt`에 이미 정리되어 있던 디자인 기준선. 함께 기록해 한 곳에서 확인 가능하게 함.

- **domitur.pt/destinations**: 전체 디자인 베이스라인 — 웜그레이 배경, 얇고 우아한 세리프 헤드라인, 얇은 테두리의 pill 라벨 + 지도 위치를 가리키는 얇은 직선 리더라인. pill은 두껍게 채우지 않고 얇은 테두리 위주.
- **for-living.it**: 지도 텍스처 참고 — 실제 경계선으로 나뉜 단색 지도, 마커는 점(core) + 얇은 링(블러/네온 글로우 없음), 지도 배경에 아주 옅은 가로선 텍스처.
- **aquamaremarine.com**: 절제된 산세리프 타이포그래피 톤 — UI 라벨/본문에 깔끔한 그로테스크 폰트, 버튼은 얇은 테두리의 차분한 스타일.
- **명시적으로 피해야 할 것**: 무거운 그림자/무거운 채움/올캡스-트래킹의 "AI SaaS" pill 버튼, 줌/범례 같은 지도 툴 UI 크롬, 흔한 크림+테라코타 "AI" 색상 팔레트.
